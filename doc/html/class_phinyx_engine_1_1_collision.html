Parameters
colliderAn instance of Collision of the body that is colliding with this body.
directionA SFML Vector2f to which the new direction of the colliding body will be written to.
forceThe force that the colliding body exercises on the collider. Forces above 0.0f will cause the colliding body to move the collider. Forces below 0.0f will cause the collider to move the colliding body, exercising a knockback effect on the colliding body.

For example, a force of 0.0f could be used when a player is colliding with a wall, as the player should not be able to move through walls.

Returns
Returns a boolean indicating if the two bodies have collided.
